Overview

Since Intraventricular haemorrhage (IVH) is one of the common complications after various acute brain injuries, leading to severe mortality and mobility, with great burdens on economics and society, we develop such a 3DAsIVH webservice that not only offers online cell subtype identification to identify a cell subtype, but also provides similarity algorithm for cell subtypes, search for similar cell types by CellMeSH, cell-cell communication and signalling pathway analysis to analysis the development of IVH.

Cell subtype identification

We used the top 30 upregulated/downregulated 3D transition genes for each 3D subtrajectory as input. Then, we employed our developed algorithm to identify a cell subtype corresponding to the selected trajectory at different times. View details

Cell subtype similarity computing

We used the marker gene sets of cell subtypes corresponding to the selected trajectory at different times as input. Then, we employed a similarity algorithm for cell subtypes to describe the similarity between cell subtypes at different times on the same trajectory. View details

Similar cell type search

We used marker genes of the cell subtypes as input. Then, we found the top 5 similar cell types for each cell subtype by using CellMeSH. Next, we performed a union operation for these top five similar cell types. View details

Cell-cell communication analysis

We used Zscore for each ligand-receptor of cell subtypes and for all the brain regions as the input. Then, we employed the cell-cell communication strength (Density) algorithm to obtain the discrete interaction intensity after conversion of the continuous interaction intensity and the sum of the interaction intensity of N points in the corresponding position, and the average for each ligand receptor of the cell subtypes.
